# GNU GDB
Some notes about GNU GDB.

## Languages supported by GDB

* Ada
* Assembly
* [C](/programming/ccplus/)
* [C++](/programming/ccplus/)
* D
* Fortran
* [Go](/programming/go/)
* Objective-C
* OpenCL
* Modula-2
* Pascal
* Rust
