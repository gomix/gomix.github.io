*** 

<img src="../images/primary-data-type.png" class="w-50 mr-5 rounded float-right" alt="C Data Types"> 
Before you go and declare a variable, you must know what kind of value will it hold.


* Primary data types:
  * char
  * int
  * float
  * double


* Modifiers:
  * unsigned
  * signed
  * long
  * short
  * const
